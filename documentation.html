Documentation

Simple documentation for the simple product

Philosophy

Replacer contains 16 HTML pages, built with only native Bootstrap 4 features, this means no additional CSS classes or JS code. You get clean and native Bootstrap layouts, with prebuilt HTML components to make it super easy to build clean and beautiful landing pages.

To master Replacer you only need official BS4 documentation, that you can find here - https://getbootstrap.com/docs/4.0/getting-started/introduction/

Icons

Icons are a crucial part of web design, so we prepared over 1000 icons in 8 palettes, and clean SVG code, to make this amazing collection that is super useful for creating web pages of any type and are easy to customize.

To add icon use the to file like this (where /secondary/ is a style of icon):src="icons/secondary/interface-32.svg

All Icons are specially designed for Replacer and strictly used together within the template

Fonts

Using Replacer you will have a huge collection of typography options for your project, all fonts are available for free, are easy to setup and supported by  Google.Fonts
